# Portfolio Project 🌐

## Overview 📎
This is a personal portfolio project built using **React** and **Vite**. The portfolio showcases my education, software proficiency, projects, and contact details, providing an organized and interactive way to present my professional journey.

---

## Features 🚀
- **Education**: Displays my academic background and achievements.
- **Software's Known**: Lists the software tools and platforms I am proficient with.
- **Projects**: Highlights my key projects with brief descriptions.
- **Contact Details**: Provides an easy way to get in touch.

---

## Technologies Used 📜
- **Frontend Framework**: React (with Vite for faster build and development)
- **Styling**: CSS / TailwindCSS (optional, based on your preferences)
- **Icons**: React Icons (for an enhanced UI/UX)
- **Hosting**: Deployed using  Vercel 

---
